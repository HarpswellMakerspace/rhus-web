ABOUT RHUS:


Rhus is an open source geospacial software stack created by WinterRoot. The software consists of native android and iphone apps, used primarily for collecting data, linked via couchdb replications with a web application, which is used for analysis and sharing. The mobile apps are designed to support offline data collection when necessary, i.e. reliable wifi or 3g is not a requirement for using the mobile applications. For more information or if you are interested in collaborating, please check out our wiki.

ABOUT WINTERROOT:


WinterRoot is a collaborative effort to bridge the gap between ecology and technology. See where we are headed at our blog, facebook, or twitter.
